# Projeto e-commerce - Tecnologia em Desenvolvimento de Sistemas
Projeto final desenvolvido para a matéria de Tecnologia em Desenvolvimento de Sistemas - CC UTFPR

## Tecnologias utilizadas;
ASP.NET,
Entity Framework,
Mysql,
HTML,
CSS,
Bootstrap,
Javascript,
RazorPages, 
Docker.

## Utilização;
Acesse a pasta raiz onde está o arquivo "docker-compose-yml, dê o comando "docker-compose up --build -d" e acesse a aplicação pela URL "http://localhost:5232/"
